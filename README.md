# Voting system
### Feature
* Decentralized voting system on top of Ethereum
* Be a host by applying for a membership
* Membership Plan
  * As a CITIZEN paying for 1 ETH
    * A max number of 10 times creating a poll
  * As a DIAMOND paying for 10 ETH
    * **Unlimited**
